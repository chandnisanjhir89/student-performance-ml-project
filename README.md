# Student Performance Analysis using Machine Learning

This project analyzes student performance using Machine Learning and Streamlit.

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Logistic Regression
- Decision Tree Classifier
- Interactive Streamlit Dashboard

## Technologies Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn
- Streamlit
